# Under Maintenance Blogger Template

A stylish, modern, and animated "Under Maintenance" page template for Blogger/Blogspot websites. This template uses CSS3 animations and a glassmorphism effect to give your site a premium look while it is being updated.

## Features
- **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop.
- **Animated Background:** Smooth moving gradient background.
- **Glassmorphism Effect:** Modern frosted glass look for the main card.
- **Floating Animation:** The content box floats gently.
- **No Images Required:** Built entirely with HTML & CSS.
- **SEO Friendly:** Meta tags included.

## How to Use
1. Download the `under-maintenance-blogger.xml` file from this repository.
2. Go to your **Blogger Dashboard** > **Theme**.
3. Click on the arrow next to "Customize" and select **Edit HTML**.
4. Delete **all** existing code.
5. Copy and paste the code from `under-maintenance-blogger.xml`.
6. Click **Save**.
